# arduino-dht11-flask
Project connecting an Arduino Uno to a DHT11 Temperature/Humidity sensor and displaying it to localhost using flask. 

### Pre-requisites 📋

```
Arduino UNO
Arduino IDE / Arduino Online IDE
DHT11 Humidity sensor
Jumper Wires
Breadboard (optional)
```
